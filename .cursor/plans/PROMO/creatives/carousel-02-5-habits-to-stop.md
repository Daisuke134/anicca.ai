# カルーセル02: 今夜やめるべき5つの習慣

## 基本情報

| 項目 | 詳細 |
|------|------|
| **タイトル（EN）** | "5 Habits Killing Your Sleep Tonight" |
| **タイトル（JP）** | 「今夜やめるべき5つの習慣」 |
| **カテゴリ** | ネガティブリスト型（Negative List Hook） |
| **スライド数** | 6枚 |
| **形式** | 静止画カルーセル（Instagram向け） |
| **アスペクト比** | **4:5（推奨）** / 1:1（任意） |
| **プラットフォーム** | Instagram Feed, Instagram Stories, Threads |
| **サムネイル/カバーテキスト** | "5 Habits Killing Your Sleep" / 「睡眠を壊す5つの習慣」 |
| **投稿ツール** | Buffer（一括予約投稿） |
| **制作ツール** | Nano Banana（画像生成） |

---

## なぜこのカルーセルが効果的か

1. **ネガティブリストの注目度** - 「やめるべき」リストは「やるべき」リストより注目されやすい
2. **具体的で実践的** - 5つの習慣は誰でも理解できる明確な行動
3. **保存されやすい** - リスト形式は「後で見返したい」と保存される
4. **視覚的なインパクト** - 「❌」マークで「やめるべき」を強調
5. **ポジティブ転換** - 最後のスライドで「✅ Aniccaがサポート」に転換し、希望を与える
6. **スワイプ誘発** - 6枚のスライドで最後まで見せる設計
