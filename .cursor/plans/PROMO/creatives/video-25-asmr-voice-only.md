## コンセプト

### なぜASMR動画が効果的か

1. **トレンド適合性** - TikTokトップトレンドの一つ（ASMR/Satisfying videos）
2. **差別化** - 情報過多のフィードで「静寂」が逆に目立つ
3. **体験型マーケティング** - 「Aniccaってどんな感じ？」を30秒で体感させる
4. **高保存率** - ASMRコンテンツは「また聴きたい」で保存されやすい
5. **感情的接続** - 声だけで「この声に毎朝起こされたい」と思わせる

### ASMRマーケティングのベストプラクティス（調査結果）

| 要素 | ポイント |
|------|---------|
| **背景音楽** | なし（完全に声のみ、または極微量のアンビエント） |
| **音質** | クリア、近接感、温かみ（ASMR特有のtingling sensation） |
| **映像** | ミニマル、抽象的、または完全黒画面 |
| **テキスト** | 最小限、邪魔にならない程度 |
| **感情的効果** | 安心、落ち着き、信頼感を創出 |

## 秒数別構成

| 秒数 | 映像 | 音声（Aniicha） | テキスト |
|------|------|----------------|----------|
| **0-3秒** | 黒画面 + Hook | 無音 | Hook: 「目を閉じて。聴いて。」 |
| **3-8秒** | 黒画面、わずかにグラデーション開始 | 「...ねぇ。...聴こえる？」 | なし |

## 音声生成仕様

### 技術要件

| 項目 | 仕様 |
|------|------|
| **サンプルレート** | 48kHz |
| **ビットレート** | 320kbps+ |
| **ダイナミクス** | ソフト、コンプレッション控えめ |
| **リバーブ** | なし〜極微量 |
| **EQ** | 高域を少しブースト（明瞭感向上） |
| **ノイズ** | 完全除去 |

### AI音声生成時の注意点

- **速度**: 通常より20-30%遅く
- **トーン**: 囁きモード、低めのピッチ
- **呼吸音**: 自然な呼吸音を含める
- **ポーズ**: 長めの間を意識
