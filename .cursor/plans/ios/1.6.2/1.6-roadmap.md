| "Meeting reminder" | **Nudge**（ミーティングに参加できない苦しみを減らす） |

## プラットフォーム別「苦しみ検出 → Nudge」の具体像

### 現在（1.6.1）

| Platform | 苦しみ検出 | Nudge | 実装 |
|----------|-----------|-------|------|
| **iOS App** | ユーザー登録の ProblemType | パーソナライズ通知 | Railway Cron |
| **Moltbook** | @mention, m/endingsuffering | compassion 返信 | OpenClaw |
| **Slack** | - | meeting reminder のみ | OpenClaw |
| **X** | - | 一方的な wisdom 投稿 | GitHub Actions |
| **TikTok** | - | 一方的な動画投稿 | GitHub Actions |

### 将来（1.7.0+）- 全て OpenClaw に統一

| Platform | 苦しみ検出方法 | Nudge 例 | OpenClaw Skill |
|----------|---------------|---------|----------------|
| **iOS App** | Server-side detection | Push 通知 | `app-nudge` |
| **Moltbook** | キーワード + センチメント | compassion 返信 | `moltbook-responder` |
| **Slack** | チャンネル監視 + キーワード | 情報提供、ヘルプ、リマインダー | `slack-helper` |
| **X** | メンション + キーワード | wisdom 返信 | `x-responder` |
| **TikTok** | コメント検出 | wisdom 返信 | `tiktok-responder` |
| **Discord** | チャンネル監視 | ヘルプ、情報提供 | `discord-helper` |
