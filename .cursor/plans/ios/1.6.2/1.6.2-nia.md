## なぜ Anicca に必要か

### 問題: 今の Nudge/Wisdom 生成の限界

| 現状 | 問題点 |
|------|--------|
| LLM の学習データに依存 | 2023年以前の情報で古い |
| Web fetch で論文を取得 | truncate されて要点を見逃す |
| 仏教テキストの引用 | 「それっぽい」でっち上げが混ざる |
| 心理学研究の参照 | 存在しない論文を自信満々で引用 |

### 例: 今起きてるハルシネーション

```
ユーザーの苦しみ: 「先延ばしがやめられない」

現状の Anicca 応答:
「心理学者の Dr. Smith が 2022年の研究で示したように、
先延ばしの根本原因は自己批判です...」

↑ Dr. Smith も論文も存在しない。LLM が作った。
```

---

## Nia で何が変わるか

### 1. **実在する仏教テキストからの引用**

```bash
# accesstoinsight.org をインデックス済み
./scripts/sources-grep.sh "accesstoinsight" "procrastination dukkha"
```

**結果:**
```
Dhammapada, verse 116: "Make haste in doing good..."
Anguttara Nikaya 8.2: "Not putting off is the cause of..."
```

**Nudge に反映:**
> 「法句経116偈にあるように、善いことを急げ。怠ると心は悪を楽しむようになる。」
> — Dhammapada 116

### 2. **実在する心理学論文からの Evidence**

```bash
# arXiv 論文をインデックス済み
./scripts/papers-index.sh "2301.08745"  # Procrastination and Self-Regulation
./scripts/search-universal.sh "procrastination self-regulation intervention"
```

**結果:**
```
"Steel (2007) found that implementation intentions reduce procrastination by 25%..."
"Self-compassion interventions show 40% reduction in chronic procrastinators..."
```

**Nudge に反映:**
> 「研究によると、『いつ・どこで・何をするか』を決めておくだけで、先延ばしは25%減る（Steel, 2007）。今日、1つだけ決めてみて。」

### 3. **感情分析データセットで苦しみ検出精度向上**

```bash
# HuggingFace emotion dataset をインデックス
./scripts/datasets-index.sh "dair-ai/emotion"
```

**Moltbook の投稿:**
> 「もう無理。何やっても続かない。」

**Nia + 感情分析:**
- sadness: 0.85
- anger: 0.12
- → ProblemType: `self_loathing` + `procrastination`
- → 最適な Nudge テンプレートを選択
