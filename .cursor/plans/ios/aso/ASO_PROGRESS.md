## 今すぐやること（並列可能）

### タスク1: Mixpanel ASA連携
**目的**: どのキーワードが課金ユーザーを連れてくるか追跡（#51の核心）

**手順**:
1. iOSアプリでApple Search Ads Frameworkを追加
2. アトリビューショントークン取得
3. Mixpanelにキーワード情報を送信
4. install → trial_start → purchase のファネルをキーワード別に追跡

### タスク2: App Store Connectでタイトル/サブタイトル変更
**目的**: キーワードを前方に配置（#52）

**手順**:
1. App Store Connect → アプリ → ローカライズ可能な情報
2. タイトル: `AI Habit Coach - Anicca`
3. サブタイトル: `AI Voice Coach & Wake Alarm`
4. 提出

### タスク3: アプリ内イベント作成
**目的**: App Store検索露出 + レビュー迅速化（#57）

**手順**:
1. App Store Connect → アプリ内イベント → 新規作成
2. イベント名にキーワード含める
3. **開始日を提出後1時間に設定**（レビュー迅速化の裏技）
