# カルーセル03: Aniccaの7つの機能

## 基本情報

| 項目 | 詳細 |
|------|------|
| **タイトル（EN）** | "7 Features That Make Anicca Different" |
| **タイトル（JP）** | 「Aniccaの7つの機能」 |
| **カテゴリ** | 機能紹介型（Feature Showcase Hook） |
| **スライド数** | 7枚 |
| **形式** | 静止画カルーセル（Instagram向け） |
| **アスペクト比** | **4:5（推奨）** / 1:1（任意） |
| **プラットフォーム** | Instagram Feed, Instagram Stories, Threads |
| **サムネイル/カバーテキスト** | "7 Features That Make Anicca Different" / 「Aniccaの7つの機能」 |
| **投稿ツール** | Buffer（一括予約投稿） |
| **制作ツール** | Nano Banana（画像生成） |

---

## なぜこのカルーセルが効果的か

1. **機能の明確化** - Aniccaの価値を7つの機能で具体的に伝える
2. **保存されやすい** - 機能リストは「後で見返したい」と保存される
3. **視覚的な理解** - 各機能をビジュアルで表現し、理解しやすく
4. **スワイプ誘発** - 7枚のスライドで最後まで見せる設計
5. **差別化ポイント** - 他アプリとの違いを明確に示す
6. **CTAへの自然な流れ** - 最後のスライドでダウンロードを促す

---

## デザイン一貫性ガイドライン

### カラーパレット
- **背景色（スライド1）**: ウォームゴールド (#f59e0b)
- **スライド2-6**: 各機能に合わせた自然な背景（朝の光、夜の光など）
- **スライド7（CTA）**: ウォームゴールド、ソフトホワイト、明るいトーン
- **テキスト**: 白、読みやすさ重視

### タイポグラフィ（後からVEED等で追加）
- **メインタイトル**: 太字、サンセリフ（Montserrat, Inter, または類似）
- **機能番号**: 数字 + 機能名（例: "1. Wake Nudge"）
- **説明文**: レギュラー、小さめのサイズ
- **フォントサイズ**: 4:5アスペクト比で読みやすく（最小24pt以上）

### レイアウトパターン
- **スライド1**: 中央配置、ミニマル
- **スライド2-6**: 人物+アプリ画面が主役、テキストは上部
- **スライド7**: 人物+アプリ画面+CTA、テキストは中央
- **余白**: 十分な余白を確保（Instagramのテキストオーバーレイ用）

### ビジュアルスタイル
- **一貫したキャラクター**: 全スライドで同じ人物を使用
- **ムードの変化**: 各機能に合わせたムード（朝の希望、夜の平和など）
- **アプリ画面**: 実際のスクリーンショットを使用
