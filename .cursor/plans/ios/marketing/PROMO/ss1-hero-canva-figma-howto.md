## このドキュメントの目的

`ss1-hero-spec.md` の仕様どおりに、**Canva / Figma のどちらでも迷わず**「1枚目スクショ（JP/EN）」を作れる手順を残す。

参照（仕様）: `.cursor/plans/PROMO/ss1-hero-spec.md`

---

## 事前準備（共通）

- **Canvasサイズ**: `1290 × 2796 px`
- **素材（朝日）**:
  - 検索キーワード例: `sunrise window` / `morning light window` / `sunrise silhouette window`
  - 条件: 「文字が乗っても邪魔しない」「過度に情報量が多くない」
- **カラー**: 水色グラデ（推奨値は仕様参照）

---

## Canva で作る（最短ルート）

### 1) 新規デザイン作成

1. Canva → 「カスタムサイズ」
2. **幅 1290 / 高さ 2796** → 作成

### 2) 背景グラデを作る

1. 「要素」→ `gradient` 検索
2. **縦グラデ**を選択
3. 色を以下に寄せる（完全一致でなくてOK）:
   - 上: `#BFEFFF`
   - 下: `#6EC9FF`

### 3) 朝日（窓）素材を重ねる

1. 「素材」→ 写真（またはアップロード）
2. 窓 + 朝日素材を配置（中央〜右寄り推奨）
3. 調整:
   - **透明度 12〜25%**
   - 必要なら「編集」→ *コントラストを下げる / ハイライトを上げる* などで“うっすら”にする

### 4) 見出しテキストを置く（JP/EN）

1. 「テキスト」→ 見出し追加
2. 文字:
   - JP:
     - 1行: `夜更かしが、`
     - 2行: `やめられる。`
   - EN:
     - 1行: `Finally stop`
     - 2行: `staying up late.`
3. スタイル:
   - 太字（Bold〜Black）
   - 中央揃え
   - 行間: やや詰め
4. 位置:
   - **画面中央やや上（上から35%付近）**
   - **四辺 96px 以上の余白**を確保

### 5) 読めるようにする（重要）

背景が明るいので、どちらか必ず入れる:

- **白文字 + 影**（推奨）
  - 影: 黒 / 20〜30% / ぼかし大きめ
- **白文字 + うっすら暗いスクリム**
  - テキスト背面に、黒 10〜18%の長方形（角丸は任意）

### 6) ロゴ（任意）

1. 右下に小さく
2. 見出しより目立たない（主役はコピー）

### 7) 書き出し

1. 「共有」→「ダウンロード」
2. **PNG / サイズ 1x（元サイズ）**
3. ファイル名:
   - `ss1_benefit_jp_1290x2796.png`
   - `ss1_benefit_en_1290x2796.png`

## Figma で作る（再現性重視）

### 1) フレーム作成

1. New design file
2. Frame → 右パネルで **W=1290 / H=2796**

### 2) 背景（グラデ）

1. Frame の Fill を Linear に変更
2. 色を以下に寄せる:
   - `#BFEFFF` → `#6EC9FF`
3. 方向: 上→下

### 3) 朝日素材を配置

1. Place image で素材を配置
2. 透明度: 12〜25%
3. 必要なら:
   - Fill mode: Crop / Fit を調整
   - Blend: Screen / Overlay を試し、**文字が最優先**になるように薄める

### 4) スクリム（推奨）

1. 見出しの背面に Rectangle を追加
2. Fill: `#000000` / 10〜18%（または上部だけ暗いグラデ）
3. Blur: 0〜12（任意）

### 5) 見出しテキスト

1. Text を追加（中央揃え）
2. JP / EN をそれぞれ別フレーム or Variantで管理
3. 太字（Bold〜Black）＋行間少し詰め

### 6) 10%チェック（必須）

1. Zoom 10%にして、読めるか確認
2. 読めない場合:
   - フォントサイズを上げる
   - 影 or スクリムを強くする
   - 朝日素材をさらに薄くする

### 7) Export

1. 右パネル Export → PNG
2. 1x
3. ファイル名は仕様通り

---

## “完成”の合格基準（共通）

- **10%表示で読める**
- 文字が背景（朝日の中心）に乗って潰れていない
- 余白 96px が守れている
- JP/ENで同じトーン・同じレイアウト（言語だけ差し替え）
