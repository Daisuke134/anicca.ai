# Mobbin Design Reference - UI改善計画

## 概要

このディレクトリには、Mobbin CommunityのFigmaデザインを参考にしたUI改善計画を保存します。

**参考URL**: https://www.figma.com/design/ofMTWHWaKsIFuuAP1vhvoT/Mobbin--Community-?node-id=2-2&m=dev

## 保存ファイル

- `design-references.md`: 各デザインの詳細な参考資料（**新規追加**）
- `figma-links.md`: Figmaデザインへの直接リンク集（**新規追加**）
- `design-analysis.md`: デザイン分析と改善提案
- `implementation-plan.md`: 実装計画

## 既存UIの現状

### 主要な画面

1. **SessionView (TalkTabView)**
   - シンプルなVStackレイアウト
   - 中央に大きなボタン
   - 上部にタイトル「Anicca」

2. **SettingsView**
   - 標準的なList形式
   - セクションごとに情報を整理

3. **HabitsTabView**
   - シンプルなナビゲーション構造
   - HabitsSectionViewを表示

4. **OnboardingFlowView**
   - 各ステップが独立したView
   - シンプルなVStackレイアウト

## 改善の方向性

Mobbinのデザインパターンを参考に、以下の点を改善予定：

- より洗練されたレイアウト
- モダンなUIコンポーネント
- 視覚的な階層の改善
- アニメーションとトランジション
- カラーパレットとタイポグラフィの改善
