# 引き継ぎ書：Pulse型プロアクティブAIアプリ「Nudge」プロジェクト

---

## プロジェクト概要

**目的:** ChatGPT Pulseを参考に、通知に具体的な価値を含むプロアクティブAIアプリを作成。最初は個人開発者/インディーハッカー向けに特化し、後にAniccaにも統合。

**アプリ名（仮）:** Nudge

**コンセプト:** ユーザーのゴールと問題を把握し、AIが最適なタイミングで「次にやるべきこと」を通知。通知に具体的な価値（例：バイラルフック）を含め、タップしなくても価値がわかる。

---

## ペルソナ

**ターゲット:** 個人開発者/インディーハッカーで、自分のアプリをTikTok/ショートフォームでプロモートしたいが、何を投稿すべきか分からず、毎日X/TikTok/YouTubeをドゥームスクロールして時間を無駄にしている人

**具体的な特徴:**
- 自分のアプリを$10k MRRにスケールしたい
- TikTokアカウントはあるが動画が伸びない
- 何がバイラルするか分からない
- トレンドを追うためにSNSを無限スクロールしている
- 気づいたら1時間経っていて自己嫌悪
- まさに成田大祐（ユーザー本人）

**なぜこのペルソナか:**
- Steven Cravottaの教え #1: 「自分の問題を解決する」→ これは自分たちの問題そのもの
- 狭すぎず広すぎない範囲
- 将来的に広げることは簡単

## 3つの質問への回答（コードを書く前に完璧に答える）

### 1. 誰のためか

**個人開発者/インディーハッカーで、自分のアプリをTikTok/ショートフォームでプロモートしたいが、何を投稿すべきか分からず、毎日X/TikTok/YouTubeをドゥームスクロールして時間を無駄にしている人**

### 2. どんな痛みを解決するか

**「バイラルなフックを探すためにX/TikTok/YouTubeをドゥームスクロールする苦しみ」を解決する**

**具体的な痛み:**
- TikTokでトレンドを探すために毎日30分〜1時間スクロール
- Xで成功者のツイートを追うために無限スクロール
- 探しているものが見つからないまま時間が溶ける
- 「今日も何も投稿できなかった」という自己嫌悪
- 情報を取りに行くのが苦痛

**解決方法:**
- 情報があなたのところに来る（取りに行かなくていい）
- 通知に具体的なフックが書いてある（タップしなくても価値がわかる）
- 1日3回、最適なタイミングで通知
- タップすると1画面で全部わかる

### 3. なぜ今すぐ買うのか

**「今日この瞬間もドゥームスクロールで時間を無駄にしているから」**

- 毎日1時間 × 30日 = 30時間/月を無駄にしている
- その30時間で動画30本作れる
- 動画30本のうち1本バイラルすれば$1,000+のインストール
- つまり、このアプリを使わないことで毎月$1,000以上の機会損失
- 月額$10〜$30で30時間と$1,000を取り戻せる

## ChatGPT Pulseの調査結果

**Pulseとは:**
- 2025年9月25日にOpenAIがリリースしたプロアクティブAI機能
- 毎晩ユーザーのチャット履歴、メモリ、Gmail/Calendarを分析
- 毎朝5-10個のパーソナライズされた視覚的カードを配信
- thumbs up/downでフィードバック → 翌日以降が改善
- 「Curate」機能で「明日こういう情報が欲しい」とリクエスト可能

**Pulseの問題点:**
- 通知だけでは内容が分からない（タップしないと何のことか不明）
- Pro tier限定（月$200）
- Pulseを単独で取り出したアプリは市場に存在しない ← これがチャンス

**Pulseからパクるべきところ:**
- thumbs up/down のフィードバック機能
- 視覚的なカード形式
- 「Curate」で明日の内容をリクエストできる
- メモリ機能（過去の会話を覚えている）

**改善する点:**
- 通知に具体的な価値（フック）を入れる（Pulseはこれができていない）
- 毎朝固定ではなく最適なタイミングに通知
- 1日10個ではなく1日3個に絞る
- 1つを「これがベスト」と明示する（優柔不断な人のため）

## アプリコンセプト

**ワンライナー:**
> あなたのゴールを知っているAIが、毎日プロアクティブに「次にやるべきこと」を通知してくる

**コアフロー:**

```
オンボーディング（チャット形式）
    ↓
「あなたのゴールは？」→ 「アプリを$10k MRRにしたい」
「今の状況は？」→ 「TikTokアカウントあるけど動画伸びない」
「何で苦しんでる？」→ 「何を投稿していいか分からない」
    ↓
バックエンド（毎日夜中に処理）
- EXAでトレンド調査
- 競合アプリのバイラル動画を分析
- 「このフック使ってみたら？」を生成
    ↓
通知（朝 or 夕方、最適なタイミング）
「"POV: My AI decides what content I should post" - このフックが今バズってる。使ってみる？」
    ↓
タップ → 1画面のカード
- トレンドの説明（簡潔に）
- 推奨フック5つ（1つは「これがベスト」と明示）
- 「やる/やらない」ボタン
    ↓
「やる」→ 褒める + 進捗記録
「やらない」→ 「なぜ？」聞く → 次回改善
```

**重要なポイント:**
- 通知にフック自体を入れる（タップしなくても価値がわかる）
- 1日3回、最適なタイミングで通知（連続しない、最低2時間空ける）
- 1画面で完結（長い記事を読む必要なし）
- やるかやらないかの2択（決断の負荷ゼロ）

## Aniccaとの関係

**理解したこと:**
- Nudge（新アプリ）= 個人開発者向けに特化したバージョン
- Anicca = 最終的にはNudgeと同じ方向に進む（能動性重視）
- 音声セッション = 補助的な機能に格下げ
- 通知 → 音声セッションではなく → 1画面カードに行く

**Aniccaの進化方向:**

**現状（受動的・アラーム的）:**
```
設定した時間 → 通知 → 音声セッション
```

**目指す姿（能動的・プロアクティブ）:**
```
AIが最適なタイミングを判断 → 通知（フック/アクション入り） → 1画面カード
```

**具体的な変更案:**
1. Habitタブ → Goalsタブに変更
   - 3ヶ月のゴール
   - 理想と問題（プロファイルから移動）
2. 通知タップ後の行き先を変更
   - 現在: 音声セッション
   - 変更後: 1画面カード（やる/やらない）
3. Behaviourタブ削除（すでに進行中）
4. 音声セッション = Talkタブで残す
   - 相談したい時に自分から行く場所
   - 通知からは行かない

**最終的な方向性:**
- 能動性が最重要（音声より優先）
- オンボーディングでゴール・理想・問題を聞く
- それを踏まえてAIがプロアクティブに通知
- 例: 「夜更かしやめたい」→ 24時45分に「もうそろそろ寝ようよ」と通知
- 例: 「TikTokでプロモートしたい」→ 「このフック今バズってる。使う？」と通知

## TikTok動画検証の具体的な内容

**検証アプローチ:** モックアップ → TikTok動画（アプリ作る前）

**理由:**
- 「建てる前に『どうやって共有するか』を計画する」（Max Blade）
- モックアップなら1-2日、MVPだと2週間
- バイラルしたらフィードバック元にMVP作る方が効率的

**作るべきモックアップ（Figmaで作る画面）:**

**画面1: ロック画面の通知**
```
┌─────────────────────────────────┐
│  Nudge                    now  │
│  「"POV: My AI decides what    │
│  content I should post"        │
│  - このフックが今バズってる。  │
│  使ってみる？」                │
└─────────────────────────────────┘
```

**画面2: 1画面カード（通知タップ後）**
```
┌─────────────────────────────────┐
│        今日のフック            │
│                                │
│  "POV: My AI decides           │
│   what content I should post"  │
│                                │
│  📈 過去24時間で12動画がこの   │
│     フックでバイラル           │
│                                │
│  ┌─────────┐  ┌─────────┐      │
│  │  使う   │  │ 別のを見る │    │
│  └─────────┘  └─────────┘      │
└─────────────────────────────────┘
```

**画面3: 「使う」タップ後**
```
┌─────────────────────────────────┐

│                                │
│   Nice! 今日これで動画作ろう   │

│   今週の投稿: 3/7              │
│   ██████░░░░                   │
│                                │
└─────────────────────────────────┘
```

**TikTok動画のスクリプト:**

**動画1: 「AIが毎朝TikTokの投稿内容を教えてくれる」**
```
フック (0-3秒): "I stopped doomscrolling for content ideas"
問題 (3-5秒): "I used to spend hours on TikTok and X trying to find what's viral"
解決策 (5-15秒): [iPhoneの画面を見せる]
- 通知が来る瞬間（ロック画面）
- 通知をタップ → 1画面カード
- 「使う」をタップ → 褒める画面
CTA (15-18秒): "Link in bio. No more doomscrolling."
```

**動画2: 「優柔不断だからAIに決めてもらう」**
```
フック (0-3秒): "I'm too indecisive so I let AI decide my next move"
問題 (3-5秒): "I spend hours thinking about what to do next. Nothing gets done."
解決策 (5-15秒): [アプリ画面を見せる] "This app gives me ONE task per day. AI already picked the best option."
CTA (15-18秒): "Link in bio"
```

**動画3: 「1日1タスクだけ」**
```
フック (0-3秒): "This app only gives me ONE task per day"
問題 (3-5秒): "Other apps give you 50 todos. You do none of them."
解決策 (5-15秒): [アプリ画面を見せる] "One notification. One action. That's it."
CTA (15-18秒): "Link in bio"
```

**成功指標:**

| 指標 | 当たり判定 |
|------|-----------|
| 3秒視聴維持率 | 50%以上 |
| 完了率 | 30%以上 |
| 保存数 | 50以上 |
| コメント | 「どこでダウンロードできる？」が複数 |
| App Store遷移 | 1本の動画から10件以上 |
