## 使用方法

1. **Gemini** (https://gemini.google.com/) にアクセス
2. **Thinkingモード**（Nano Banana Pro）を選択
3. このファイル + `images/` フォルダ内の画像を渡す
4. 各スライドのプロンプトをコピーして実行

## プロンプトのベストプラクティス

### スタイルキーワード

白黒・ミニマルスタイルの場合：
- `monochrome`, `black and white`, `minimal`
- `clean line art`, `silhouette style`
- `white background`, `no decorations`
- `simple composition`, `centered`

### アスペクト比

- **16:9**: プレゼンテーション標準
- **1:1**: アイコン・アクセント用

### 注意点

1. **テキストを入れない**: `No text in the image` を必ず指定
2. **シンプルに**: 複雑な指示より簡潔な指示の方が良い結果
3. **参照画像がある場合**: `Keep the original exactly as provided` を追加
4. **一貫性**: 全スライドで同じスタイルトーンを維持
